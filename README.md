## <a href="https://roky-test.vercel.app/" target="_blank">Roky</a>

ToDo-приложение, позволяющее управлять текущим списком дел

## Технологии

* React
* Redux-toolkit
* TypeScript
* SCSS
* Material UI

Clone repositories:

```
https://github.com/Nikita1025/roky-test.git
```
Install dependencies:

```
npm install
```
Start project:

```
npm start
```
