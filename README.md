## <a href="https://todo-mu-livid.vercel.app/" target="_blank">Todo</a>

ToDo-приложение, позволяющее управлять текущим списком дел

## Технологии

* React
* Redux-toolkit
* TypeScript
* SCSS
* Material UI

Clone repositories:

```
https://github.com/Nikita1025/todoTest.git
```
Install dependencies:

```
npm install
```
Start project:

```
npm start
```
